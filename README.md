# Lost in Transmission: An Analysis of GNSS Robustness

GNSS architectures are continuously challenged by external signal interference and malicious threats. 
This report aims to analyze positioning systems' resilience in various contexts, from ideal scenarios to environments with potential sources of interference, as well as the effects of simulated spoofing scenarios. 
The obtained results suggest that measurements can vary greatly among different GNSS chipsets even in ideal scenarios, while physical obstacles and spoofing attacks can effectively undermine GNSS resilience.

This repo provides the dataset for the Laboratory activity on GNSS for the Wireless Security in the Cybersecurity Engineering MSc @ PoliTo
